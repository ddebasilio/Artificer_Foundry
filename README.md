# Artificer Foundry - Alchemy & Crafting

A comprehensive alchemy and crafting tool for Foundry VTT. Features a recipe book for potions and items, ingredient management (from common herbs to rare monster parts), crafting time tracking, and a drag-and-drop crafting UI.

## Features

- **Recipe Book**: Manage recipes for potions, items, and more.
- **Ingredient Management**: Track various ingredients like herbs, monster parts, and minerals.
- **Crafting UI**: A user-friendly drag-and-drop interface for crafting.
- **Foraging System**: Built-in foraging system with biome and abundance modifiers.
- **Time Tracking**: Track crafting time.

## Installation

1. Open Foundry VTT and go to the Add-on Modules tab.
2. Click "Install Module".
3. Paste the following Manifest URL: `https://raw.githubusercontent.com/ddebasilio/Artificer_Foundry/refs/heads/main/module.json`
4. Click "Install".

## Usage

Once installed and enabled in your world, you can access the Artificer Foundry tools from the Actor sheet or via macros. The module provides interfaces for gathering ingredients and crafting items based on defined recipes.

## Compatibility

- **Minimum Foundry VTT version**: 12
- **Verified Foundry VTT version**: 14
